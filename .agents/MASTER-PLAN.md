# Hacıyatmaz Kablo — Master Büyüme Planı
*Oluşturulma: 2026-03-30 | Stack: ui-ux-pro-max + 21st.dev + Stitch + Nano Banana + Marketing Skills*

---

## Teşhis: Neden 0 Satış?

Sorun reklam değil. **Kırık dönüşüm hunisi.**
Reklam tıklaması sayfaya geliyor → sayfa güven vermiyor → kullanıcı terk ediyor.

### Kök nedenler:
1. H1 zayıf — acı noktası yok, kim için olduğu belirsiz
2. Fiyat tutarsızlığı (821 TL vs 799 TL) → güven kaybı
3. En güçlü USP (stand) sayfa altında gömülü
4. CAPS LOCK yorgunluğu → spam izlenimi
5. Marketplace CTA'ları hero'da kaybolmuş
6. Sosyal kanıt (yorum/yıldız) görünür değil
7. Reklam → dönüşmeyen sayfa = para yakma

---

## Strateji: 3 Faz

```
FAZ 1 (Hafta 1-2)   → Conversion Fix: Sayfa dönüştürmeden reklam verme

```
